# ROS Stereo Image Publisher with e-CAM24_CUNX cameras

## Build

1. Build `libcapture.so` as stated at [Here](CapCapture/README.md)
2. Put this repository into `~/catkin_ws/src` directory
3. Run `$ catkin_make` at `~/catkin_ws`
